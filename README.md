# finance-dashboard
This project contains the source and other components for a Python based personal financial dashboard backed by a data ingestion and processing engines. Deploy it, drop in CSVs of credit card, bank account statements and visualize your financial health. 

## Key Faeatures
- Ingestion of statements of credit card/bank accounts in csv format
- 90% of rules defined to identify expense types
- Identify credit/debit or spend/earn entries and categorise accordingly
- A protected UI dashboard based on the data
- More ...
